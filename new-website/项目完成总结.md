## 🚀 使用方法

### 本地查看
1. 直接在浏览器中打开 `index.html`
2. 或使用本地服务器：
   ```bash
   # Python
   python -m http.server 8000
   
   # Node.js
   npx http-server
   ```

### 部署
将所有文件上传到Web服务器即可，使用相对路径，无需特殊配置。
