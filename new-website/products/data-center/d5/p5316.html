D5-P5316

Efficient, read-optimized PCIe 4.0 SSD with capacities up to 30.72TB, designed for read-intensive and sequential workloads in data centers.

Product Overview

The Solidigm D5-P5316 is a highly efficient, read-optimized PCIe 4.0 NVMe SSD that delivers industry-leading storage efficiency for read-intensive and sequential workloads. With capacities up to 30.72TB and support for E1.L and U.2 form factors, it is ideal for cloud storage, data lakes, content repositories, and other large-scale data center deployments.

Key Features

  • Read-optimized design for read-intensive and sequential workloads
  • Capacities up to 30.72TB for dense storage deployments
  • PCIe 4.0 x4 NVMe interface
  • E1.L and U.2 15mm form factors for flexible integration
  • High storage efficiency with QLC 3D NAND
  • Ideal for cloud storage, data lakes, and content repositories

Technical Specifications

Interface PCIe 4.0 x4, NVMe
Form Factors E1.L, U.2 15mm
Capacity Options 15.36TB, 30.72TB
Sequential Read (up to) 7,000 MB/s
Sequential Write (up to) 3,600 MB/s
Random Read IOPS (4KB, up to) 1,000,000 IOPS
Random Write IOPS (4KB, up to) 80,000 IOPS
Endurance Read-intensive
NAND Type QLC 3D NAND
Power Consumption (Active) Up to 14W
Operating Temperature 0°C to 70°C
MTTF 2.0 million hours
