D5-P5336

Optimized for the edge to the core, the D5-P5336 is a hyper-scalable, cost-effective solution that lets you efficiently accelerate data. Industry-leading 122TB capacity.

Product Overview

The Solidigm D5-P5336 represents a breakthrough in storage density and efficiency. With industry-leading capacities up to 122.88TB, this QLC SSD delivers exceptional value for read-intensive, data-intensive, and mainstream workloads. Optimized for applications from edge computing to core data centers, it maximizes space, power, and capacity efficiency.

Key Features

  • Industry-leading 122.88TB capacity
  • PCIe 4.0 x4 NVMe interface
  • Capacities from 7.68TB to 122.88TB
  • U.2 15mm, E3.S 7.5mm, and E1.L 9.5mm form factors
  • Hyper-scalable and cost-effective
  • Optimized for edge to core applications
  • Power-efficient QLC technology

Technical Specifications

Interface PCIe 4.0 x4, NVMe 2.0
Form Factors U.2 15mm, E3.S 7.5mm, E1.L 9.5mm
Capacity Options 7.68TB, 15.36TB, 30.72TB, 61.44TB, 122.88TB
Sequential Read (up to) 7,200 MB/s
Sequential Write (up to) 3,800 MB/s
Random Read IOPS (4KB, up to) 1,200,000 IOPS
Random Write IOPS (4KB, up to) 200,000 IOPS
Endurance Read-intensive (0.58 DWPD)
NAND Type QLC 3D NAND
