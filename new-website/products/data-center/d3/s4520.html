D3-S4520

Standard-endurance SATA SSDs up to 7.68TB, designed to increase server efficiency and boost IOPS/TB performance over legacy HDDs for cloud data storage.

Product Overview

The Solidigm D3-S4520 is a standard-endurance SATA SSD that offers up to 7.68TB of capacity, providing a compelling replacement for legacy HDDs in cloud and enterprise storage. It increases server efficiency and delivers significantly higher IOPS per terabyte than mechanical drives, while maintaining familiar SATA infrastructure.

Key Features

  • Standard-endurance SATA SSD with capacities up to 7.68TB
  • SATA 3.0 6Gb/s interface
  • 2.5" 7mm form factor for easy HDD replacement
  • Higher IOPS/TB versus legacy HDDs
  • Optimized for cloud data storage and general server workloads
  • Enterprise-grade reliability

Technical Specifications

Interface SATA 3.0 6Gb/s
Form Factor 2.5" 7mm
Capacity Options 480GB, 960GB, 1.92TB, 3.84TB, 7.68TB
Endurance Standard endurance
Operating Temperature 0°C to 70°C
