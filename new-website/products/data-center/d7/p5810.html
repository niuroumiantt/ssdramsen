D7-P5810

Discover the Solidigm D7-P5810, the ultra-fast PCIe 4.0 SSD at 800GB and up to 50 DWPD for workloads with extreme write intensity and high endurance needs.

Product Overview

The Solidigm D7-P5810 is engineered for extreme write-intensive workloads that demand the highest levels of endurance and performance. With up to 50 DWPD (Drive Writes Per Day) and PCIe 4.0 connectivity, it delivers exceptional reliability for mission-critical applications requiring continuous high-intensity data writing.

Key Features

  • Ultra-high endurance up to 50 DWPD for extreme workloads
  • PCIe 4.0 x4 NVMe interface for maximum performance
  • Optimized for write-intensive applications
  • U.2 15mm form factor for enterprise compatibility
  • Advanced endurance management and data protection
  • Built for mission-critical, high-reliability environments

Technical Specifications

Interface PCIe 4.0 x4, NVMe
Form Factors U.2 15mm
Capacity Options 800GB, 1.6TB, 3.2TB
Sequential Read (up to) 7,000 MB/s
Sequential Write (up to) 4,100 MB/s
Random Read IOPS (4KB, up to) 1,100,000 IOPS
Random Write IOPS (4KB, up to) 400,000 IOPS
Endurance High (up to 50 DWPD)
NAND Type TLC 3D NAND
Power Consumption (Active) 14W (max)
Operating Temperature 0°C to 70°C
MTTF 2.5 million hours
